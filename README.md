# 🕷️ AranhaVerso 🕷️

> Live da DIO: Criando um carrossel parallax do AranhaVerso com React, Next.js 13 e Framer Motion

## Technologies

- Typescript (v5)
- React Js (v18)
- Next Js (v13)
- Tailwind CSS
- Framer Motion

## How to Run

### Requisites

- Node.js

### Run

- clone repository with git
- npm install
- npm run dev
- access url http://localhost:3000
